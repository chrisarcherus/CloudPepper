POS Bluemax

Integrate your POS with Bluemax Payment Terminal

Overview

This module provide integration between odoo pos and bluemax pay.

Configuration

Need to install global payments package before we're installing this module. Go to the Point of Sale > Configuration > payment methods. Set up a new payment method for bluemax.

Features

  • Integrate Odoo with BlueMax Pay.
  • Cashless transaction.
  • Enable payment option for particular POS Session.
