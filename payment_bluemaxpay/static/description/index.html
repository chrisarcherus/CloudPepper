BlueMax Pay Payment

BlueMax Pay Payment acquirer

Overview

This module provide integration between odoo pos and bluemax. This module allows customers to make the payment using their credit card. Without the presence of credit card, using the card details the customer's can make our payments. Creating new merchant user when onboarding a new user.

Configuration

Need to install global payments package before we're installing this module.
Go to the Point of Website > Configuration > Configuration settings. Set up a new payment type for bluemax.

Features

  • Integrate Odoo with BlueMax Pay.
  • Cashless transaction.
  • By using the card details customers can make the payment.
  • We can use Both Authorize and Capture
