BlueMax Pay Card Not present Payment

BlueMax Pay Payment acquirer Card not present method

Overview

This module provide integration between odoo pos and bluemax. This module allows customers to make the payment using their credit card. Without the presence of credit card, using the card details the customer's can make our payments.

Configuration

Need to install global payments package before we're installing this module. Go to the Point of Sale > Configuration > payment methods. Set up a new payment method dor bluemax.

Features

  • Integrate Odoo with BlueMax Pay.
  • Cashless transaction.
  • By using the card details customers can make the payment
  • Enable payment option for particular POS Session.
