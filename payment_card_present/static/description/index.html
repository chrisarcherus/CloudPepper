Overview

This module provide integration between odoo Invoice and bluemax. This module allows customers to make the payment using their credit card. Without the presence of credit card, using the card details the customer's can make our payments.

Configuration

Go to the Invoicing > Configuration > Settings > Bluemax Pay. Add the credentials.

Features

  • Integrate Odoo with BlueMax Pay.
  • Cashless transaction.
  • By using the card details customers can make the payment

Screenshots

1. Bluemax Pay Configuration

Go to the Invoicing > Configuration > Settings > Bluemax Pay. Add all the Credentials.

2. Invoice

Go to Invoicing > Payment > Create the Payment and Select the payment method as Bluemax Pay Card present.

3. Bluemax Pay Payment

Click the Bluemax Pay Payment Button It opens a Popup.

4. Bluemax Pay Payment

Click the Send Payment Request. Then confirm the payment
