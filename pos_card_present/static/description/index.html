ZilloPay Card present Payment

ZilloPay Payment acquirer Card present method

Overview

This module provide integration between odoo pos and zillopay. This module allows customers to make the payment using their credit card. Without the presence of credit card, using the card details the customer's can make our payments.

Configuration

Go to the Point of Sale > Configuration > payment methods. Set up a new payment method dor zillopay.

Features

  • Integrate Odoo with ZilloPay.
  • Cashless transaction.
  • By using the card details customers can make the payment
  • Enable payment option for particular POS Session.
